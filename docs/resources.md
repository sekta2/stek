# 📚 Документация >> 📦 Ресурсы
Система ресурсов, тут она оптимизированней чем в JMod, вместо строк в STek используется числа, это нужно для оптимизации ведь строки занимают хорошую долю в размере сетевого пакета.

## 🛠️ Функции

### stek.add_resource(индетификатор)
Добавляет новый ресурс, так же создаёт с ним новую глобальную переменную для удобства, например **STEK_RESOURCE_STEEL**

- индетификатор - строка(string) - steel

### stek.get_resource(индекс)
Возвращает индетификатор ресурса под указанным индексом

- индекс - число(number) - 1
