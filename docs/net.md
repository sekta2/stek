# 📚 Документация >> 📣 Сеть
Есть пара функций полезных в оптимизации сети.
В основном тут только функции которые помогают определить сколько бит нужно на определённое число.

## 🛠️ Функции

### net.WriteSmartInt(число)
Отправляет умное "Integer"

- число - число(number) - 1337

### net.ReadSmartInt()
Читает умное "Integer"

### net.WriteSmartUInt(число)
Отправляет умное "Unsigned Integer"

- число - число(number) - 1337

### net.ReadSmartUInt()
Читает умное "Unsigned Integer"

## ❓ Отличие UInt от Int
UInt - "Unsigned Integer" - Неподписанное *целое число*, тоесть от 0 до 32 бита(**4,294,967,295**)

Int - "Integer" - *целое число*, поддерживает и отрицательные числа, тоесть от **-2,147,483,648** до **2,147,483,647**.
