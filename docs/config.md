# 📚 Документация >> ⚙️ Конфигурация
В аддоне есть только один конфиг, и он находится по пути `lua/stek/sh/config.lua`, и рекомендуется добавлять переменные с пояснениями и в категориях.

## 📜 Методичка
Например у нас есть супер-пупер крутой верстак, и для него нужны переменные для конфигурации, чтобы можно было в одном файле и в одном месте быстро изменить что либо.

```lua
--[[------------------------
    Super-Puper Workbench
]]--------------------------

-- Какая то важная переменная 1
STEK_CFG_SPW_VIV1 = 1

-- Какая то важная переменная 2
STEK_CFG_SPW_VIV2 = 1337

-- Какая то важная переменная n
STEK_CFG_SPW_VIVn = 0
```
