# ⚒️ STek (Sekta Tech)
Аддон, база для всех JMod-подобных аддонов.

## Философия/Концепция
STek подразумивается как база для всех JMod-подобных аддонов, предоставляя базовые инструменты, энтити и модули.

### Структура аддона/контент пака
```
lua/
    stekpacks/
        pack_name/
            locale/
                ru.lua
                en.lua
            resources.lua
            gas.lua
```
- структура не финальная, по мере разработке возможны большие изменения в структуре
