# 📚 Документация >> 🗂️ Действия
Чтобы не создавать 1,000 net'ов для различных действий была создана система "действий".

## 🛠️ Функции

### stek_actions.add_action(название, задержка, функция)
Добавляет действие

- название - строка(string) - scrounge
- задержка - число(number) - 5
- функция - функция(function) - function(**ply**) end
- функция:ply - игрок

### stek_actions.exec_action(игрок, название)
Выполняет действие игрока

- игрок - энтити(entity)
- название - строка(string) - scrounge

## 📝 Где добавлять
Советую добавлять действия в скрипте `lua/stek/sh/actions.lua` в самом низу
