## 🛠️ Функции

### stek_util.set_get_function(мета-таблица, название, ключ/переменная, стандартное-значение: НЕ ОБЯЗАТЕЛЬНО)
Добавляет мета-таблице функции по типу SetHealth и GetHealth, используется в stek-squads и является заменой AccessorFunc

- мета-таблица - таблица(table)
- название - строка(string) - Health
- ключ/переменная - строка(string) - health

- стандартное-значение - любое(any) - 100
